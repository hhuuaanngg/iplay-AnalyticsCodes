You can customize wp_header and wp_footer simple HTML code plug-in.
可以自定义wp_header和wp_footer部的HTML代码的简单插件。

== Description ==
A very simple custom code plug-in, developers intention was to add their own web statistics code. After easily add custom CSS and HTML code for the head and tail.
Thank you for using.
一个非常简单的自定义代码插件，开发原意是为了自己添加网站统计代码。之后顺手添加了自定义CSS和头部和尾部的代码。
谢谢使用。

== Installation ==
1. Upload the folder "c-html" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
